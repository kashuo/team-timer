# Team Timer

A simple app for soccer coaches to track player playing time during games. Built as a Progressive Web App — install it on your phone's home screen and use it like a native app.

## Features

- Multiple concurrent stopwatches, one per player
- Players sorted by least playing time (so you know who to sub in)
- Tap a name to edit it
- "New Game" saves current times to season totals and resets
- Season totals screen shows cumulative time across all games
- Works offline — no internet needed at the field
- Data persists on your device between sessions

## Install on iPhone

1. Open [kashuo.github.io/team-timer](https://kashuo.github.io/team-timer/) in **Safari**
2. Tap the **Share** button (square with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **Add**

## Install on Android

1. Open [kashuo.github.io/team-timer](https://kashuo.github.io/team-timer/) in **Chrome**
2. Tap the **three-dot menu** (top right)
3. Tap **"Add to Home screen"** or **"Install app"**
4. Tap **Add**

## Local Development

```
python3 -m http.server 8000
```

Open http://localhost:8000 in your browser.
